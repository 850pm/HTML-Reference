HTML Form Reference

Tags

Tag Name Description
<form> Form Defines a HTML form for user input
<input> Input Specifies an input field where the user can enter data
<select> Select Is poulated with <option> elements and renders a dropdown list selection
<datalist> Datalist Is poopulated with <option> elements and works within an <input> to search through choices
<textarea> Textarea A text input field that has a customizable area

Attributes

Attribute Name Description
<action> Action Specifies where to send the form-data when a form is submitted
<method> Method Specifies the HTTP method to use when sending form-data
<text> Text Creates a single row field for text input
<password> Password Creates a single row field that censors text input
<number> Number Creates a single row field for number input
<range> Range Creates a slider to select from a range of numbers
<checkbox> Checkbox Creates a single checkbox which can be paired with other checkboxes
<radio> Radio Creates a radio button that can be paired with other radio buttons
<list> List Will pair the <input> with a <datalist> element if the id of both are the same
<submit> Submit Creates a submit button
<required> Required Specifies that the element must be filled out before submitting the form
<min> Min Specifies a minimum value
<max> Max Specifies a maximum value
<minlength> Min Length specifies the minimum number of characters required in an input field
<maxlength> Max Length Specifies the maximum number of characters allowed in an element
<pattern> Pattern Specifies a regular expression that an <input> element's value is checked against
